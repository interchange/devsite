Development code

Source code repository (Git)

You can browse the source code and see changes made to it over time using our Gitweb site or at our GitHub repository. You can also use a Git client to download a read-only copy of the repository to your computer: 

git clone git://git.icdevgroup.org/interchange

Interchange core developers with commit access can clone a writable copy: 

git clone ssh://$USER@git.icdevgroup.org/var/git/interchange

Everyone else is welcome to contribute to the project by forking our GitHub repository, committing changes to their own GitHub fork, and making a pull request to us. Or send us a patch. Either way, discuss your changes with us on the interchange-users mailing list.
