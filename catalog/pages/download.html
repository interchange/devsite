Stable releases

__IC__ is primarily run on x86_64 Linux distributions Debian, Ubuntu, RHEL/â€‹CentOS, openSUSE, Fedora, Arch, and Gentoo. It has also been used on i386 Linux, FreeBSD, OpenBSD, NetBSD, Solaris, macOS, and some more obscure Unixes.

The current release of __IC__ is __STABLE__, available in cross-platform package interchange-__STABLE__.tar.gz. Previous releases are also available.

PGP/GnuPG keys of Interchange developers are available to validate digital signatures and communicate privately.

Platform-specific Debian and RPM packages are no longer actively maintained. We recommend installing from the above cross-platform package.

Git source code repository

You can browse the source code and see changes made to it over time in Git using our GitHub repository. You can also use a Git client to download a copy of the repository to your computer: 

git clone https://github.com/interchange/interchange.git

That latest automatic development version build is at interchange-nightly.tar.gz, and past __NIGHTLY_HTTP__ are available.

Everyone is welcome to contribute to the project by forking our GitHub repository, committing changes to their own GitHub fork, and making a pull request to us. Or send us a patch. Either way, discuss your changes with us on the interchange-users mailing list linked from the [page users/community]community page.
