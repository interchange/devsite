The current stable release of Interchange is __STABLE__.

CVS

You can browse the latest __STABLE_CVSWEB__ CVS tree online using __CVSWEB__.
You can also use a cvs client to download a read-only copy from CVS to your computer: 

cvs -z6 -d :pserver:cvs@cvs.icdevgroup.org:/var/cvs \
  checkout -P -r STABLE___STABLE_CVS_SERIES__-branch interchange

Debian GNU packages

Packages for the __DEBIAN__ operating system are being maintained by __racke__, a Debian developer. You can find the packages and apt-get setup instructions in a separate __STABLE_DEBIAN__ (you can also browse the top-level __DIR_DEBIAN__ for other versions).

RPM packages

Packages for Red Hat-derived operating systems are being maintained by __jon__. You can find the packages in a __STABLE_RPM__ and __STABLE_SRPM__ directories.
